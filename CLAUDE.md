# RULES

- 不要使用npm，使用pnpm

# 介绍

这是一个视频流共享(watch party)平台

- 1. 房间：用户可以创建房间，加入房间，一个房间只有一个房主

- 2. 房间通话：同一个房间的用户可以点击发起通话创建/加入到通话中

- 3. 视频流：房主或者获得权限的人，可以放映视频流，所有房间内的成员都可以看到

- 4. 视频同步：所有房间内的人，都会同步视频的进度

- 5. 视频进度条拖动：房主或者获得权限的人，可以拖动视频的进度条。被拖动的进度条因为（4）的原因，会被同步到房间内的其他人

- 6. 视频源：要能够支持多个视频源，比如夸克网盘，bilibili视频。最重要的是，不应该让用户自己下载然后上传，应该让用户提供一个视频源的定位符，网站会自动解析并且提供视频流服务。
